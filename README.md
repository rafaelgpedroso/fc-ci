# Integração Contínua com GitHub Actions e SonarCloud

## 📌 Objetivo
Este projeto tem como objetivo consolidar conhecimentos sobre **CI (Continuous Integration)** e **Qualidade de Código**, configurando uma pipeline completa para garantir que nenhum código seja integrado à branch `main` sem antes:

- passar nos testes automatizados
- passar pela análise de qualidade do **SonarCloud**
- respeitar o **Quality Gate**

---

## 🛠 Tecnologias utilizadas
- Node.js
- Jest (Testes unitários)
- GitHub Actions (Pipeline CI)
- SonarCloud (Análise de qualidade)
